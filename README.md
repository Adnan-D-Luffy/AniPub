## üöÄ Get Started Instantly

**1. Clone the repo**
```bash
git clone https://github.com/AdnanDLuffy/AniPub.git
cd AniPub
```

**2. Install dependencies**
```bash
npm install
```

**3. Launch it locally**
```bash
node backend/app.js
# Then open http://localhost:3000 in your browser
```
